# TaskHelper Bot

Телеграм-бот для управления задачами.

## Функции
- Добавление задач: /add <задача>
- Просмотр задач: /list
- Отметка выполненных: /done <номер>
- Удаление задач: /delete <номер>

## Технологии
- Python
- PyTelegramAPI
- SQLite
